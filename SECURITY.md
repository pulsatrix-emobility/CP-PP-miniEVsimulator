# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository or in any pulsatrix product, please report it responsibly:

- **Email:** security@pulsatrix.de
- **Response time:** we acknowledge every report within 3 business days
- **Coordinated disclosure:** we do not publish an advisory without prior coordination with the reporter
- **Full policy:** see our [Coordinated Vulnerability Disclosure Policy](https://pulsatrix.de/security) for scope, embargo terms, and what to expect

Please do not open public GitHub issues for security vulnerabilities.

## Scope

This repository is part of the pulsatrix product suite, which falls under the EU Cyber Resilience Act. See the CVD Policy linked above for the full list of in-scope products and components.

## Sicherheitsrichtlinie

## Schwachstelle melden

Wenn du eine Sicherheitslücke in diesem Repository oder einem anderen pulsatrix-Produkt entdeckst, melde sie bitte verantwortungsvoll:

- **E-Mail:** security@pulsatrix.de
- **Reaktionszeit:** Eingangsbestätigung innerhalb von 3 Werktagen
- **Koordinierte Offenlegung:** wir veröffentlichen kein Advisory ohne vorherige Abstimmung mit dem Finder
- **Vollständige Policy:** siehe unsere [Coordinated Vulnerability Disclosure Policy](https://pulsatrix.de/security) für Scope, Embargo-Regeln und was du erwarten kannst

Bitte keine öffentlichen GitHub-Issues für Sicherheitslücken anlegen.

## Geltungsbereich

Dieses Repository ist Teil der pulsatrix-Produktfamilie, die unter den EU Cyber Resilience Act fällt. Die vollständige Liste der betroffenen Produkte und Komponenten findest du in der oben verlinkten CVD Policy.
